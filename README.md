# post-reflection-deliberative-ai-tudelft
This repository contains the prompts, anonymised transcripts, evaluation codebook, and results used in the experiments presented in the associated paper, supporting reproducibility and further research on LLM-based reflective dialogue generation and evaluation.

It was created for the purposes of the 2026 iteration of the [TUDelft CSE3000 Research Project](https://github.com/TU-Delft-CSE/Research-Project) for the [CSE Bachelor](https://https//github.com/TU-Delft-CSE).
